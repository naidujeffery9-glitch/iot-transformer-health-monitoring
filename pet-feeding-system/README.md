# Pet Feeding System ??

An automated pet feeding system project.

## Features
- Automated feeding schedule
- Portion control
- Remote monitoring
- Notifications

## Setup
1. Install Python 3.8+
2. Install dependencies: `pip install -r requirements.txt`
3. Run: `python main.py`

## Hardware Requirements
- Raspberry Pi/Arduino
- Servo motor
- Food container
- Sensors
